In ```<section>``` element with a class of "gender", add two ```<label>```s that have associated ```<input>``` elements.

The created ```<input>``` should have:
1. for ```male```
* an ```id``` set to ```"male"```.
* a ```type``` set to ```"radio"```
* a ```name``` attribute with a value of ```"gender"```.
* a ```value``` of ```"male"```.
2. for ```female```
* an ```id``` set to ```"female"```.
* a ```type``` set to ```"radio"```
* a ```name``` attribute with a value of ```"gender"```.
* a ```value``` of ```"female"```.
